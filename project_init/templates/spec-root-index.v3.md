# Spec 索引

> 本文件用于快速回溯历史需求关注的模块和处理方向；节点状态的 source of truth 仍是各 spec 状态锚点。
> OMS 文档正文默认使用中文；路径、代码标识符、API 名称、frontmatter 枚举值和既有英文术语可保留英文。

## 命名规则

- Patch 单文件：`docs/spec/YYYY-MM-DD-{slug}.md`
- Feature 多文件目录：`docs/spec/YYYY-MM-DD-{slug}/index.md`
- 日期为 spec 创建日期，精确到日；后续更新不改文件名或目录名。
- `slug` 保持短小稳定，表达需求主题。
